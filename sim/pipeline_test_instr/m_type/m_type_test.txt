RV32M 乘除法指令测试
=====================

测试指令序列（每条指令后标注预期结果）：

# 设置寄存器
ADDI x1, x0, 10       # x1 = 10
ADDI x2, x0, 3        # x2 = 3
ADDI x3, x0, 20       # x3 = 20
ADDI x4, x0, 6        # x4 = 6
ADDI x5, x0, -1       # x5 = 0xFFFFFFFF

# MUL 测试
MUL  x6, x1, x2       # x6 = 10 * 3 = 30 (0x1E)
MUL  x7, x5, x2       # x7 = (-1) * 3 = -3 (0xFFFFFFFD)

# MULH 测试（有符号*有符号，取高32位）
MULH x8, x5, x2       # x8 = (-1) * 3 高32位 = -1 (0xFFFFFFFF) (sign-extended)

# MULHU 测试（无符号*无符号，取高32位）
MULHU x9, x5, x2      # x9 = 0xFFFFFFFF * 3 高32位 = 2 (0x2)

# MULHSU 测试（有符号*无符号，取高32位）
MULHSU x10, x5, x2    # x10 = (-1) * 3(无符号) 高32位 = -1 (0xFFFFFFFF)

# DIV 测试
DIV  x11, x3, x4      # x11 = 20 / 6 = 3
DIV  x12, x5, x2      # x12 = (-1) / 3 = -1 (0xFFFFFFFF)

# DIVU 测试
DIVU x13, x3, x4      # x13 = 20 / 6 = 3
DIVU x14, x5, x2      # x14 = 0xFFFFFFFF / 3 = 1431655765 (0x55555555)

# REM 测试
REM  x15, x3, x4      # x15 = 20 % 6 = 2
REM  x16, x5, x2      # x16 = (-1) % 3 = -1 (0xFFFFFFFF)

# REMU 测试
REMU x17, x3, x4      # x17 = 20 % 6 = 2
REMU x18, x5, x2      # x18 = 0xFFFFFFFF % 3 = 0

# 除零测试
ADDI x19, x0, 42      # x19 = 42（被除数）
DIV  x20, x19, x0     # x20 = 42 / 0 = -1 (0xFFFFFFFF)
REMU x21, x19, x0     # x21 = 42 % 0 = 42

# NOP 填充
NOP
NOP
NOP

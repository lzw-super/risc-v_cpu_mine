# ==============================
# RISC-V J型指令动态预测测试程序 (验证版)
# ==============================
# 功能: 测试JAL和JALR的动态跳转预测
#       验证BTB缓存跳转目标和预测效果
# ==============================

# 地址    | 机器码      | 指令                    | 说明
# --------|-------------|------------------------|------------------------------------------
#         |             | === Part 1: BNE循环计数 ===
# 0x00    | 0x00000093  | ADDI x1, x0, 0         | x1 = 0 (计数器)
# 0x04    | 0x00A00113  | ADDI x2, x0, 10        | x2 = 10 (循环上限)
# 0x08    | 0x00100193  | ADDI x3, x0, 1         | x3 = 1 (增量)
# 0x0c    | 0x003080B3  | ADD x1, x1, x3         | x1++ (循环体)
# 0x10    | 0xFE209EE3  | BNE x1, x2, -4         | BNE跳回0x0c if x1 != x2
#         |             |                        | 验证分支预测BTB缓存效果
# --------|-------------|------------------------|------------------------------------------
#         |             | === Part 2: JAL向前跳转预测 ===
# 0x14    | 0x00100213  | ADDI x4, x0, 1         | 循环结束后执行: x4 = 1
# 0x18    | 0x008002EF  | JAL x5, 8              | JAL跳到0x20, x5=返回地址0x1c
#         |             |                        | 首次执行: BTB miss, flush流水线
# 0x1c    | 0x00000013  | NOP                    | 跳过不执行
# --------|-------------|------------------------|------------------------------------------
# 0x20    | 0x05000313  | ADDI x6, x0, 0x50      | JAL目标: x6 = 0x50
# 0x24    | 0x06000393  | ADDI x7, x0, 0x60      | x7 = 0x60 (另一个JALR目标)
# --------|-------------|------------------------|------------------------------------------
#         |             | === Part 3: JALR预测测试 ===
# 0x28    | 0x000304E7  | JALR x9, x6, 0         | JALR跳到x6(0x50), x9=返回地址0x2c
#         |             |                        | 首次执行: BTB miss, flush
# 0x2c    | 0x000384E7  | JALR x9, x7, 0         | (不执行，因0x28已跳转)
# --------|-------------|------------------------|------------------------------------------
# 0x30-   | NOP填充到0x50              |
# --------|-------------|------------------------|------------------------------------------
# 0x50    | 0x00100413  | ADDI x8, x0, 1         | JALR目标: x8 = 1
# 0x54-   | NOP填充                     |
# --------|-------------|------------------------|------------------------------------------
# 0x60    | 0x06400513  | ADDI x10, x0, 100      | 从0x50顺序执行到此: x10 = 100
# --------|-------------|------------------------|------------------------------------------

# ==============================
# 实际执行流程:
# ==============================
# 1. BNE循环: x1从0累加到10，循环10次
#    - 首次BNE: BTB miss，flush
#    - 后续BNE: BTB命中，预测正确跳转
#    - 循环退出: BNE不跳转，BTB预测mispredict一次
#
# 2. 循环结束: x4 = 1
#
# 3. JAL x5, 8: 从0x18跳到0x20
#    - 首次执行: BTB miss, flush
#    - x5 = 0x1c (返回地址)
#    - NOP at 0x1c被跳过
#
# 4. 执行0x20-0x24: x6 = 0x50, x7 = 0x60
#
# 5. JALR x9, x6, 0: 从0x28跳到0x50
#    - 首次执行: BTB miss, flush
#    - x9 = 0x2c (返回地址)
#    - JALR at 0x2c不执行(已跳转)
#
# 6. 执行0x50: x8 = 1
#    继续执行NOPs直到0x60
#    执行0x60: x10 = 100
#
# ==============================
# 验证结果 (已通过):
# ==============================
# x1 = 10    ✓ (BNE循环计数正确)
# x4 = 1     ✓ (循环后执行正确)
# x5 = 0x1c  ✓ (JAL返回地址正确)
# x6 = 0x50  ✓ (JALR目标寄存器正确)
# x7 = 0x60  ✓ (JALR目标寄存器正确)
# x8 = 1     ✓ (JALR跳转目标执行正确)
# x9 = 0x2c  ✓ (JALR返回地址正确)
# x10 = 100  ✓ (顺序执行到0x60正确)
#
# ==============================
# BTB缓存验证:
# ==============================
# - BNE首次执行: BTB miss → flush
# - BNE后续执行: BTB命中 → 预测正确跳转
# - BNE退出循环: mispredict (预测跳转，实际不跳)
# - JAL首次执行: BTB miss → flush
# - JALR首次执行: BTB miss → flush
# - 总mispredict: 2次 (BNE首次 + BNE退出)
# - 总BTB miss重定向: 2次 (JAL + JALR)
